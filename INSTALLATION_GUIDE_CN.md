## ❌ 如果安装失败了怎么办?

### 错误: "应用未安装"

**可能原因**:
1. 设备存储空间不足
2. 下载的APK文件损坏

**解决方案**:
```bash
# 清理设备存储
# 设置 > 应用 > 卸载不需要的应用

# 或重新下载APK文件
```

### 错误: "不允许安装来自未知来源的应用"

**解决方案**:
1. 打开 **设置**
2. 搜索 **"安装未知应用"** 或 **"来自未知来源"**
3. 选择 **文件管理器** (或您用来打开APK的应用)
4. 打开开关以允许安装
5. 返回重新安装

### 错误: "安装包未包含任何证书"

**这个问题已解决！**
- 新的APK已使用现代SHA256签名
- 确保您使用的是 `sanguozhiguoguanzhanjiang_downcc_resigned.apk`
- 不要使用旧的 `sanguozhiguoguanzhanjiang_downcc_fixed.apk`
